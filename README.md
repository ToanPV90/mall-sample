## Prerequisites

- Java 8+

- Docker

- Docker-Compose

## Start Environment

1. Go to infrastructure folder
    ```shell script
    cd infrastructure/
    ```

2. Run command:

    ```shell script
    docker-compose up -d
    ```

3. Check environment health:

    ```shell script
    docker-compose ps
    ```
